*********************************************  1.Jacoco Tutorial (Java Code Coverage) (vogella)  *********************************************

Jacoco i.e. Java code coverage is an open source project which can be used to check production code for test code coverage.
It creates reports and integrates well with IDE's like Eclipse.
Integration is also available for other IDE's and continuous integration environments.
